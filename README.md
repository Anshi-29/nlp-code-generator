## 🌟 What is this project?

This is a **full-stack web application** that allows users to:

1. **Register & Login** securely using JWT authentication
2. **Type plain English** like *"write a for loop in Python"*
3. **Get instant executable code** in Python, JavaScript, SQL, C, or Java
4. **Copy or Download** the generated code
5. **View history** of last 6 generated codes

No need to remember syntax — just describe what you want in simple English!

---

## 🚀 Features

- 🔐 **Secure Authentication** — Register/Login with JWT tokens
- 🔒 **Password Security** — Hashed with bcryptjs
- 🤖 **NLP Engine** — English to Code conversion
- 🐍 **5 Languages** — Python, JavaScript, SQL, C, Java
- 🎨 **Animated UI** — Matrix rain + particle network background
- 📋 **Copy Code** — One click copy to clipboard
- ⬇️ **Download Code** — Save as .py .js .sql .c .java
- 🕐 **History** — Last 6 generated prompts saved
- 💾 **MySQL Database** — All users stored securely
- 📱 **Responsive** — Works on all screen sizes
